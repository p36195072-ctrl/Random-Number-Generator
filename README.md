# Random-Number-Generator
Random Number Generator

A beginner-friendly Python project that demonstrates how to generate random numbers using Python.

📌 Description

This program uses Python’s random module to generate and print a random number between 1 and 10.

Each time the program runs, a different number may appear.

🧠 Concepts Used

random module
randint() function
Variables
print() function

💻 Code

import random

number = random.randint(1, 10)

print(number)

▶️ Example Output

7
